## Youtube视频批量下载器
**仅用于学术研究使用**

### 环境
+ python2.7
+ win10 64位测试
+ 已翻墙（可正常访问Youtube等国外网站）

### 工具
+ python2.7
+ Beautiful Soup
+ urllib
+ 一个叫做[http://keepvid.com/](http://keepvid.com/ "keepvid")的网站。该网站能够提供一个Youtube链接并返回该视频的下载链接

### 使用方法
	python BatchDownloader.py ./urls/train.txt 或
	python BatchDownloader.py ./urls/test.txt
